# ADR 0001: 외부 의존성 없는 로컬 웹 애플리케이션

- 상태: 승인
- 날짜: 2026-06-08

## 배경

MVP는 서버, 계정, 데이터베이스 없이 시험지 파일을 열고 응시·채점해야 한다. 저장소에는 아직 기술 스택과 배포 방식이 정해지지 않았으며, 문서는 단순한 데이터 흐름과 결정적인 채점 결과를 우선하도록 요구한다.

## 결정

초기 구현은 최신 데스크톱 브라우저에서 실행하는 HTML, CSS, JavaScript ES 모듈로 작성한다.

- 앱 실행에는 정적 파일 서버만 사용하며 런타임 네트워크 요청을 만들지 않는다.
- 시험지는 버전이 명시된 JSON 파일로 불러온다.
- 도메인과 채점 로직은 DOM 및 파일 시스템과 분리된 순수 JavaScript 모듈로 둔다.
- 자동 검증은 Node.js 20 이상의 내장 테스트 러너를 사용한다.
- 앱 데이터는 메모리에만 보관하고 결과 내보내기는 사용자의 명시적 동작으로만 수행한다.

## 결과

외부 패키지 설치 없이 개발과 검증이 가능하고 브라우저의 파일 선택·다운로드·인쇄 기능을 재사용할 수 있다. 반면 브라우저에서 `index.html`을 직접 여는 경우 ES 모듈 보안 정책에 영향을 받을 수 있으므로 로컬 정적 서버 실행 절차를 제공한다. 데스크톱 패키징은 실제 배포 요구가 확인될 때 별도 ADR로 결정한다.
