# 보안 및 개인정보 기준

## 1. 보호 대상

- 시험 문제, 정답, 해설, 채점 기준이 포함된 시험지 파일
- 진행 중 답안과 제출 스냅샷
- 채점 결과와 선택적으로 입력된 응시자 정보
- 결과 파일이 저장된 로컬 기기와 이동식 저장매체

## 2. 보안 범위

이 제품은 네트워크, 계정, 서버를 사용하지 않는 단순 오프라인 응시 도구다. 서버 공격, 세션 탈취, API 인가, 온라인 동기화는 위협 범위에 포함하지 않는다. 대신 로컬 파일 노출, 제출 전 정답 표시, 결과 파일의 개인정보 과다 수집을 우선 통제한다.

## 3. 핵심 위협과 통제

| 위협 | 통제 |
| --- | --- |
| 제출 전 정답 노출 | 응시 화면 모델에서 정답·채점 규칙 제외, UI 계약 테스트 |
| 잘못된 시험지로 오채점 | 파일 스키마, 문제 ID, 배점 합계, 정답 형식 사전 검증 |
| 채점 결과 변동 | 결정적인 정규화·채점 함수와 고정 테스트 벡터 사용 |
| 결과 파일 개인정보 노출 | 기본 익명 결과, 선택한 식별 정보만 포함, 저장 전 미리보기 |
| 로컬 파일 무단 열람 | 운영체제 계정·파일 권한 사용, 공유 기기 저장 자제 안내 |
| 악성 또는 과도한 입력 파일 | 파일 크기, 문항 수, 문자열 길이 제한과 안전한 파서 사용 |
| 로그를 통한 답안 노출 | 답안 원문, 정답, 개인정보를 기본 로그에 기록하지 않음 |
| 결과 파일 변조 | 필요성이 확인되면 해시 또는 전자서명 추가 검토 |

## 4. 시험지와 정답 보호

- 시험지 파일은 신뢰할 수 있는 시험 준비자가 생성한 파일만 연다.
- 파일 확장자만 신뢰하지 않고 실제 구조와 스키마 버전을 검증한다.
- 응시 화면에 전달하는 모델에는 정답과 내부 채점 규칙을 포함하지 않는다.
- 제출 이벤트 전에는 정답·해설 표시 함수를 호출할 수 없도록 상태를 검사한다.
- 같은 로컬 파일에 정답이 존재하는 구조는 강한 비밀 보호를 보장하지 않음을 운영 문서에 명시한다.

## 5. 데이터 최소화와 보관

- 응시에 계정이나 로그인을 요구하지 않는다.
- 응시자 이름, 번호 등 식별 정보는 결과 저장에 필요한 경우에만 선택적으로 입력한다.
- 현재 답안은 다른 시험지를 불러오거나 같은 시험 새 응시를 명시하기 전까지 메모리에 유지하고 채점 결과는 시험 ID별 누적 CSV로 브라우저 저장소에 기록한다.
- 디렉터리 선택과 쓰기 권한 요청은 사용자가 시험 준비 화면에서 명시적으로 실행한다. 제출 시에는 권한 요청이나 저장 창을 열지 않고 승인된 디렉터리의 사번별 누적 CSV만 갱신하며, 권한이 없으면 브라우저 저장과 수동 저장을 유지한다.
- 공유 기기에서는 결과 파일을 남기지 않거나 시험 시작 화면의 CSV 백업 후 삭제 기능으로 현재 버전 기록을 제거한다. 전체 삭제는 시험 기록 접두사에 해당하는 키만 제거한다.
- 보관 기간은 조직의 품질·개인정보 규정으로 정하며, 제품은 무단 자동 삭제 대신 건수·용량 경고와 명시적 삭제를 제공한다.
- 임시 파일을 만들 경우 정상 종료와 오류 종료에서 삭제되는지 검증한다.

## 6. 로컬 기록

운영 진단이 필요할 때 다음 최소 이벤트만 선택적으로 기록한다.

- 시험지 검증 성공·실패와 오류 코드
- 응시 시작, 제출, 채점 완료 상태
- 결과 내보내기 성공·실패
- 애플리케이션 오류와 버전

답안 원문, 정답, 해설, 응시자 식별 정보는 로그에 기록하지 않는다.

## 7. 부정행위 방지 한계

- 이 도구는 감독, 브라우저 잠금, 화면 녹화, 온라인 신원 확인을 제공하지 않는다.
- 개발자 도구, 파일 직접 열람, 운영체제 조작을 강한 보안 경계로 차단하지 않는다.
- 강한 시험 보안이 필요한 경우 감독 환경, 암호화된 시험 배포, 별도 온라인 시스템을 검토해야 한다.
- 단순 교실 평가, 자가 학습, 연습 시험처럼 로컬 운영이 적합한 사용 사례를 기본 대상으로 한다.

## 8. 보안 검증

- 제출 전 응시 화면에 정답 필드가 포함되지 않는 계약 테스트
- 잘못되거나 과도하게 큰 시험지 파일에 대한 입력 검증 테스트
- 동일 입력에 대한 채점 결과 결정성 테스트
- 경로 조작과 의도하지 않은 파일 덮어쓰기 테스트
- 결과 파일 개인정보 기본값과 미리보기 검증
- 배포 전 비밀 및 불필요한 샘플 개인정보 스캔
